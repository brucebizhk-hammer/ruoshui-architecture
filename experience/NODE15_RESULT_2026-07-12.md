# Node 15 Result - Owner DM Review Commands

Status: complete

## Delivered

- Owner can confirm with `确认群提炼 <记录号>` or `群提炼确认 <记录号>`.
- Owner can reject with `驳回群提炼 <记录号> <原因>`.
- Owner can correct with `修正群提炼 <记录号> <JSON对象>`.
- Commands are accepted only from an allowlisted owner direct message.
- Group messages and non-owner direct messages cannot execute review commands.
- Every command uses the Node 14 append-only review workflow and sends a private success/failure receipt.
- Ambiguous text and invalid correction JSON do not alter the database.
